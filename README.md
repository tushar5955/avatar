# Avatar Project

This is a Vite + React TypeScript project for building customizable avatars with cybernetic features.

## Features
- Modular avatar components (face, eyes, data particles)
- Easily extendable for new avatar features
- Modern React + TypeScript codebase
- Vite for fast development and builds

## Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure
```
avatar-app/
  src/
    components/
      Avatar.tsx
      cybernetic/
        CyberneticFace.tsx
        CyberneticEyes.tsx
        DataParticles.tsx
    App.tsx
    main.tsx
  public/
  styles/
  package.json
  vite.config.ts
```
